# GDSC Member

⚠️ This project needs more optimization, like component reusing etc.

## Requirement
- Nodejs >= 16
- Package manager like `npm` or `pnpm`

## Installation
```
pnpm install
```
And create an `.env` for your api endpoint
```
VITE_API_ENDPOINT=************
```

## Development
```
pnpm run dev
```

## Build for production
```
pnpm run build
```
